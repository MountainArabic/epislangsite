Обзор

EpisLang — язык, написанный на Python 3.12.9, ориентированный на точные математические вычисления. Используйте ссылки выше для подробной документации.

Язык EpisLang ориентирован на студентов и ученых

Для студентов - для их обучения мы предоставляем удобный и понятных синтаксис для построения алгоритмов, а так-же решение всех математических проблем пайтона, что-бы новички не заморачивались с проблемами IEEE754 и с внешними гигантскими модулями для точных чисел
Для ученых - мы предоставляем короткий но структурированный синтаксис языка и точные типы, а также работаем над новым модулем formula.py для еще более улучшенной работой с данными и научными данными
